# 🎸 ChordBuddy: JavaScript Music Theory Engine

An interactive, open-source music utility and algorithmic chord library built with **React** and **JavaScript**. This project serves as a technical reference for developers building complex, state-driven audio-visual interfaces and music-related software.

---

## 🛠️ Technical Implementation (For Developers)

This project is specifically relevant to the **JavaScript ecosystem** as it demonstrates how to manage complex musical logic and dynamic UI rendering without static databases.

### Key JavaScript Features:
* **Dynamic Scale Algorithms:** Implements modular JS functions to calculate musical intervals, scale degrees, and chord voicings algorithmically.
* **State-Driven Fretboard Mapping:** Utilizes React hooks (`useState`, `useEffect`) to manage real-time UI updates for fretboard positions with minimal overhead.
* **Declarative SVG Rendering:** Demonstrates how to render dynamic SVG elements for fretboard markers and string positions based on calculated mathematical musical data.
* **State Management:** Handles multi-layered state for tuning, chord selection, and scale transposition in a unified JavaScript logic core.

### Why this is a Developer Resource:
ChordBuddy is designed for **JS developers** building **Web Audio API** applications, MIDI-based tools, or educational music software. The architecture is modular, allowing developers to fork the logic for scale generation and UI mapping into their own JavaScript-based web applications.
